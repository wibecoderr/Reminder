# Reminder

A simple terminal-based reminder script written in **Shell**.  
This tool lets you add a date, time, and message from the terminal, and it will notify you with the message (and sound if supported) at the scheduled moment. :contentReference[oaicite:2]{index=2}

---

## 🧠 Overview

Sometimes we set tasks for the next day but forget them in the morning.  
This script solves that problem by allowing you to schedule reminders directly from the terminal. At the appointed day and time, it pops up your message and plays a sound (if available). :contentReference[oaicite:3]{index=3}

---

## 🚀 Features

- 🗓️ Add reminders with **date** and **time**
- 💬 Displays reminder message at the right time
- 🔔 Plays sound notification (if system supports it)
- 🖥️ Works on systems that support shell scripting (Linux / macOS / Git-Bash on Windows) :contentReference[oaicite:4]{index=4}
